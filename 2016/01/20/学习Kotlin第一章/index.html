###使用Kotlin开发Android

兼容性：Kotlin与JDK 6完全兼容，确保Kotlin应用程序可以在较旧的Android设备上运行而不会出现问题。
Android Studio完全支持Kotlin工具，并与Android构建系统兼容。

性能：由于非常相似的字节码结构，Kotlin应用程序的运行速度与等效的Java应用程序一样快。
由于Kotlin支持内联函数，使用lambdas的代码通常比使用Java编写的相同代码运行得更快。

互操作性：Kotlin与Java 100％互操作，允许在Kotlin应用程序中使用所有现有的Android库。
这包括注释处理，因此数据绑定和Dagger也可以工作。

足迹：Kotlin有一个非常紧凑的运行时库，可以通过使用ProGuard进一步减少。
在实际应用程序中，Kotlin运行时只添加几百个方法，并且小于.kk文件大小的100K。

编译时间：Kotlin支持高效的增量编译，因此虽然对于干净的构建有一些额外的开销，但增量构建通常比使用Java快或快。

Kotlin功能工具

Kotlin Android Extensions是一个编译器扩展，它允许您摆脱findViewById()代码中的调用，并用合成编译器生成的属性替换它们。

Anko是一个库，提供围绕Android API的一组Kotlin友好包装器，以及一个允许您使用Kotlin代码替换布局.xml文件的DSL。

基本类型

Numbers

Kotlin处理数字在某种程度上接近Java,但是并不完全相同.

对于数字没有隐式拓宽转换（如 Java 中 int 可以隐式转换为long——译者注),另外有些情况的字面值略有不同.

Kotlin 提供的内置类型来表示数字（与 Java 很相近）：
注：Kotlin 中字符不是数字

Type Bit width
Double 64
Float 32
Long 64
Int 32
Short 16
Byte 8

字面常量 数值常量字面值有以下几种:

十进制: 123
Long 类型用大写 L 标记: 123L
十六进制: 0x0F
二进制: 0b00001011

注意：不支持八进制

Kotlin 同样支持浮点数的常规表示方法:

Doubles ：123.5, 123.5e10
Floats：标记f或F：123.5f

可以使用下划线使数字常量更易读
