Main函数

main函数可以保证该的独立运行.它是程序的入口.它会被JVM所调用.

步骤：

  • 通过class关键字定义一个类。将代码都编写到该类中。
  • 为了保证该的独立运行。在类中定义个主函数。格式public static void main(String[] args)
  • 保存成一个扩展名为java的文件。
  • 在dos控制台中通过javac工具对java文件进行编译。
  • 在通过java命令对生成的class文件进行执行。

面向对象(main函数)

主函数：是一个特殊的函数。作为程序的入口，可以被jvm调用。

主函数的定义：

  • public：代表着该函数访问权限是最大的。
  • static：代表主函数随着类的加载就已经存在了。
  • void：主函数没有具体的返回值。
  • main：不是关键字，但是是一个特殊的单词，可以被jvm识别。
  • String[] arr）:函数的参数，参数类型是一个数组，该数组中的元素是字符串。字符串类型的数组。

主函数是固定格式的：jvm识别。

  • jvm在调用主函数时，传入的是new String[0];
